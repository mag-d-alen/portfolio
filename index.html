Hello,

I'm Magdalena

Hi, I'm Magdalena, a front-end developer, a JavaScript enhusiast and a React eager initiate, currently based in Tel Aviv, Israel.

I started programming in mid-noughties, as an MA student in Digital Humanities at University College London. Back then, web aplications were build in JavaScript-HTML-CSS, with an SQL database, using PHP for the backend.
After finishing my degree, I did a PhD and a post-doc in early modern history. About two years ago, I rediscovered my passion for the Front End--and with ES6, NodeJS, Reach and Sass--what not to love!

To keep up with the industry, I finished two intensive full-stack bootcamps (over eighty hours of coding!) and started developing my own small projects, some of which can be sampled below↓. I am looking for an oportunity to learn further and contribute to a team effort. I would love to hear from you :)

This is a NodeJS express app. It uses a time and date API to display the current date as the title of the list. Items can be added and deleted. Additional functionality allows the user to create their own list with a custom title. I used a NoSQL database (MongoDB) to store the client data.
The app is styled with CSS (using media-query for mobile design) and fully resonsive.
The app is hosted on Heroku and is currently live, so you are more than welcome to check it out.
The source code is available on Github.
The app was built to entertain my two cats, Shufi and Mi.
It is built with React.js and features a search engine and profiles of cat friends whose are obtained from a REST API and added to profile components.
It is styled with modern CSS with flexbox as an overall structure asnd fully responsive.
The code base is available on Github.
Voice Notes is a ReactJS app built in ES6 with functional components and React hooks. It was built to try out the open-source speech recognition tool, Web Speech Api.
The recording begins when the user turns on the microphon, speech is converted to text and a note is created. The note can be then saved and later deleted. The app is styled with modern CSS and fully responsive.
The source code is available on Github.
This portfolio was built and designed by me from scratch. It is mostly HTML and SCSS with a little of vanilla JavaScript (ES6) to enhance user exerience (smooth scrolling and dynamic overlay). I used CSS grid for the layout and flexbox to style components. The design takes advantage of a number of CSS3 features such as gradients and box-shadows, as well as animations, variables, and nesting of the SCSS and uses 8bit images and gif demos of a few of my projects.
The source code is available on Github.
